# 📄 Proyecto: Página Web con HTML Semántico y Despliegue en GitHub Pages

## 📌 Descripción

Este proyecto consiste en el desarrollo de una página web utilizando etiquetas HTML semánticas y buenas prácticas de estructuración del contenido.  
Además, se implementó control de versiones con Git y se publicó el sitio en línea mediante GitHub Pages.

---

## 🛠️ Actividades Realizadas

### 1️⃣ Estructura HTML Semántica

La página fue estructurada utilizando etiquetas fundamentales como:

- `html`
- `head`
- `body`
- `title`
- `meta`

Estas etiquetas permiten organizar correctamente el documento y mejorar su accesibilidad.

---

### 2️⃣ Jerarquía de Títulos

Se utilizaron etiquetas de encabezado para organizar la información:

- `h1` para el título principal
- `h2`, `h3`, etc., para subsecciones

Esto garantiza una estructura clara y lógica del contenido.

---

### 3️⃣ Párrafos

Se empleó la etiqueta:

- `p`

Para agregar contenido informativo y descriptivo dentro de la página.

---

### 4️⃣ Enlaces

Se incorporaron enlaces mediante la etiqueta:

- `a`

Estos permiten redirigir a otras páginas o recursos externos.

---

### 5️⃣ Imágenes

Se añadieron imágenes utilizando:

- `img`

Incluyendo atributos como `src` y `alt` para mejorar la accesibilidad.

---

### 6️⃣ Elementos Opcionales

Se integraron elementos adicionales para enriquecer la página:

- Listas desordenadas (`ul`)
- Formularios (`form`, `input`, `label`)

---

## 🔄 Control de Versiones

Se utilizó **Git** para:

- Registrar cambios durante el desarrollo.
- Documentar avances mediante commits.
- Mantener control sobre las versiones del proyecto.
